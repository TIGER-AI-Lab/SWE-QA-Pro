# SWE-QA-Pro Training

The training recipe has two stages, run in order:

1. **SFT**: supervised fine-tuning of Qwen3-8B on 1000 agentic trajectories generated by Claude Sonnet 4.5, using [ms-swift](https://github.com/modelscope/ms-swift).
2. **RL**: agentic reinforcement learning (GRPO) on top of the SFT checkpoint, using [verl-tool](https://github.com/TIGER-AI-Lab/verl-tool) with a repository tool server and an LLM judged rubric reward.

Run SFT first, then use its checkpoint as the base model for RL. Each stage has its own environment and a dedicated README with full setup and run instructions: [`SFT/README.md`](SFT/README.md) and [`RL/README.md`](RL/README.md).

## File structure

```
train/
├── README.md                       # This file
├── SFT/                            # Stage 1: supervised fine-tuning (ms-swift)
│   ├── README.md                   # SFT setup, data download, and training
│   ├── scripts/
│   │   └── train_sweqapro_8B.sh
│   └── dataset/                    # SFT trajectories; not in the repo, download from HuggingFace (see SFT/README.md)
└── RL/                             # Stage 2: reinforcement learning (verl-tool)
    ├── README.md                   # RL setup and training
    └── verl-tool/                  # verl-tool codebase
        ├── scripts/
        │   └── train_sweqapro_8B.sh
        ├── dataset/                # RL train/test parquets
        ├── verl_tool/              # with custom swe_qa_pro tool + reward manager
        └── verl/                   # vendored verl
```
